# PGP Key & Messages — Cryptographic Authentication

Cicada 3301's official OpenPGP public key and signed messages used for authentication across all three puzzles. The key ID 7A35090F is the canonical identifier for verifying Cicada communications.

## Contents

- **key/** — The official Cicada 3301 public key (7A35090F) in ASCII-armored format
- **messages/** — PGP-signed messages and communications authenticated with the key

## Verification

The key can be verified using the `isitcicada` web tool: [github.com/cicada-solvers/isitcicada](https://github.com/cicada-solvers/isitcicada)

All authentic Cicada communications bear this signature. The key has remained consistent across all puzzle rounds.
