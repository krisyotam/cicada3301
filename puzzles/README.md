# Puzzle Rounds — Cicada 3301 Original Puzzles

Original puzzles from 2012, 2013, 2014, and 2016 with complete walkthroughs, images, assets, and cryptographic materials. Each round represents a distinct challenge released by Cicada 3301, ranging from steganography to PGP authentication.

## Contents

- **2012/** — First puzzle (January 2012). Includes PUZZLE-2012.md overview, WALKTHROUGH.md with step-by-step solution, images of posters and clues, and puzzle assets
- **2013/** — Second puzzle (January 2013). Continuation of narrative and escalated difficulty
- **2014/** — Third puzzle (January 2014). Released Liber Primus as the culmination
- **2016/** — Fourth puzzle (2016). Recruitment puzzle with modern steganography techniques

Each year subdirectory contains the original media (images, documents) and structured documentation of the solve path.
