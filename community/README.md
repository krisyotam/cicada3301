# Community

Active communities working on Cicada 3301, primarily focused on solving Liber Primus.

## Discord
- **CicadaSolvers** — ~7,000 members, the primary active community
- Regular voice sessions for collaborative solving

## IRC
- **#cicadasolvers** on Libera.Chat — 50-100 active members
- Historical solving discussions

## Reddit
- **r/cicada** — ~21,000 subscribers
- **r/a2e7j6ic78h0j** — the original subreddit from the 2012 puzzle (archived)
