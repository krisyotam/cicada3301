# Cicada 3301 — Complete Timeline

## 2012

| Date | Event |
|------|-------|
| January 4 | First puzzle image posted to 4chan /x/ board. Black JPEG with white text: "Hello. We are looking for highly intelligent individuals..." |
| January 5–6 | OutGuess steganography extraction reveals PGP-signed message (key ID 7A35090F). Book cipher decodes to URL at 845145127.com (845145127 = 509 × 503 × 3301, where 509×503 are the image pixel dimensions) |
| January 6–9 | 845145127.com displays cicada image and countdown timer. A phone number is discovered playing a recorded message with additional clues |
| January 9 | Countdown expires, reveals 14 GPS coordinates worldwide |
| January 9–15 | Physical white paper posters with cicada stencils and QR codes found on streetlights at GPS locations: Warsaw, Paris, Seoul, Sydney, Hawaii, Miami, New Orleans, Seattle, and others |
| January 15–31 | QR codes contain book cipher references to William Gibson's "Agrippa (A Book of the Dead)" (1992). Book cipher leads to .onion URLs accessible via Tor |
| February | Winners admitted to PGP-gated forum. Assigned tasks related to privacy, cryptography, and censorship-resistant tools |

## 2013

| Date | Event |
|------|-------|
| January 4 | Second puzzle image posted |
| January–March | Bootable Linux ISO discovered (Cicada OS, ~130MB, based on Tiny Core Linux). Contains audio files and the message "@1231507051321 The key is all around you. Good luck. 3301" |
| 2013 | Gematria Primus cipher system introduced: 29 Anglo-Saxon runes mapped to prime numbers |
| 2013 | "The Instar Emergence" (761.mp3) music file discovered. 2:47 duration (167 seconds — 167 is prime and the emirp of 761). ID3 tag contains hidden poem: "Parable 1,595,277,641 / Like the instar, tunneling to the surface / We must shed our own circumferences; / Find the divinity within and emerge." |
| 2013 | Marcus Wanner (age 15) solves the second puzzle |
| 2013 | Winners invited to anonymous chat server, tasked with building a "dead man's switch" for whistleblowers |
| Late 2013 | Project falls apart. Cicada goes silent |

## 2014

| Date | Event |
|------|-------|
| January 4 | Third puzzle clue posted on Twitter @1231507051321 |
| May 2, 2014 | **Liber Primus released** — 58 pages of Elder Futhark runes. Message: "Hello. Your enlightenment awaits you. We look forward to hearing from you." |
| 2014–2015 | Community begins decoding Liber Primus. Approximately 17 pages eventually solved using combinations of Gematria Primus transliteration, Vigenère cipher with prime-based keys, Euler's totient function, and Atbash variants |
