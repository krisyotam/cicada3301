# Cicada 3301 — Complete Archive & Documentation

A comprehensive archive of everything related to Cicada 3301: the original puzzles (2012–2014), Liber Primus, community analysis, tools, and primary sources. Built as the foundation for a documentation website.

## Structure

```
puzzles/           Original puzzle walkthroughs and assets per year
liber-primus/      The unsolved book — pages, decoded content, solving tools
analysis/          Writeups, academic papers, conference presentations
tools/             Steganography, cryptanalysis, and community-built tools
resources/         Primary source materials — images, audio, ISO, PGP keys
timeline/          Chronological event documentation
website/           Future documentation site
```

## Status

Cicada 3301 posted three rounds of cryptographic puzzles (2012, 2013, 2014) before going silent. The final known PGP-signed message was April 2017. Liber Primus — a 58-page runic text released in May 2014 — remains largely unsolved (~17 of 58 pages decoded). A small community continues work via Discord, IRC, and GitHub.
