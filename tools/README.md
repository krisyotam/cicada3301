# Tools — Cryptanalysis, Steganography & Solver Utilities

Community-built tools for cryptanalysis, steganography verification, and Liber Primus deciphering. These utilities range from complete solving platforms to specialized analysis applications for rune mapping and cipher verification.

## Contents

- **solvers/** — Complete solving platforms including iBotPeaches' Laravel app, web-based Liber Primus tools, Python cryptanalysis suites, and automated solver attempts
- **cryptanalysis/** — Specialized utilities for frequency analysis, substitution breaking, and cipher identification
- **verification/** — Tools for PGP signature verification and steganography detection (including isitcicada web app)
- **TOOLS-INDEX.md** — Comprehensive catalog of 20+ GitHub repositories with descriptions, steganography tools (OutGuess, OpenPuff, stegdetect), and learning resources

## Key Tools

**Steganography**: OutGuess (JPEG extraction), OpenPuff (multi-carrier), stegdetect (detection), Sonic Visualiser (audio analysis)

**Solving**: LiberPrTools-Web, idkfa, LiberPrimusSolver, CicadaArchiveTool, isitcicada verification

**Learning**: Khan Academy cryptography, Game Detectives Academy, ae27ff puzzle training, TryHackMe Cicada room
