## Steganography Tools Used in Puzzles

| Tool | Purpose |
|------|---------|
| **OutGuess** | Primary tool for extracting PGP-signed messages from JPEG images. Developed 1999, BSD license |
| **OpenPuff** | Multi-carrier steganography tool with 3-layer obfuscation. Used to analyze Cicada images |
| **stegdetect** | Steganography detection tool by Niels Provos. Used by Joel Eriksson in initial analysis |
| **Sonic Visualiser** | Audio spectrogram viewer. Used to find hidden QR code in 3301.wav spectrum |
